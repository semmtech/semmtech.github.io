Data Sharing Agreements

Documentation of the DSA vocabulary is available here in PDF format:
  • https://w3id.org/dsa/dsa.pdf
The DSA vocabulary employs the following namespace:
  • https://w3id.org/dsa/
The DSA vocabulary can be retrieved in two ways:
  1. access its namespace with the preferred Content-Type "text/turtle" in the request
  2. use this direct link: https://w3id.org/dsa/dsa.ttl
